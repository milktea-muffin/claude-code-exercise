# PRD: [AI Todo]

## What（何を作るか）

AIが優先度を自動判定するチーム向けToDoアプリ。
Google OAuthでログインし、タスクを作成するとAIが内容・期限・コンテキストから
優先度（High/Medium/Low）を自動付与する。
チームでタスクを共有・割り当てでき、Webブラウザとモバイルブラウザで利用可能。

### 解決する課題
- タスクの優先度付けに毎日15分以上かかっている
- チーム内でタスクの優先度認識がズレて手戻りが発生する

### ターゲット
5-20人規模のスタートアップのプロダクトチーム。
日常的にタスク管理ツールを使っているが、優先度付けに不満を感じている。

## Features（機能一覧）

### MVP（これがないとリリースできない）
| # | 機能 | 説明 |
|---|------|------|
| 1 | ユーザー認証 | Google OAuth でログイン/ログアウト |
| 2 | Todo CRUD | 作成・一覧・編集・削除 |
| 3 | AI 優先度判定 | 作成時に自動で High/Medium/Low を付与 |
| 4 | チーム共有 | チーム内でTodoを共有・割り当て |

### Later（MVP後）

|---|------|------|
| 1 | Slack 通知 | 期限切れ・割り当て時に通知 |
| 2 | 繰り返しタスク | 毎日/毎週の定期タスク |
| 3 | ダッシュボード | 完了率・チーム統計 |

## Screens（画面一覧）

| 画面 | パス | 認証 | 主な要素 |
|------|------|------|---------|
| ログイン | /login | 不要 | Google OAuth ボタン |
| Todo一覧 | / | 必要 | リスト、フィルタ、作成ボタン |
| Todo詳細 | /todos/:id | 必要 | 編集フォーム、削除 |
| チーム設定 | /team | 必要 | メンバー一覧、招待 |

画面遷移: ログイン → Todo一覧 ↔ Todo詳細、Todo一覧 → チーム設定

## Constraints（制約）

- **パフォーマンス**: 初回読み込み 3秒以内
- **セキュリティ**: 認証必須、HTTPS、入力バリデーション
- **対応環境**: Chrome/Safari/Firefox 最新版、iOS Safari、Android Chrome
- **スコープ外**: ネイティブアプリ、オフライン対応、多言語対応
