# PRD（Product Requirements Document）作成

$ARGUMENTS の内容をもとに PRD を作成する。

prd-writer スキルを参照し、assets/template.md のテンプレートと
references/example.md の見本に従って docs/PRD.md を作成する。

## 手順

1. ユーザーの入力から要件を把握する
2. 不足情報はまとめて1回で質問する（1問ずつ聞かない）
3. prd-writer スキルの手順に従って PRD を作成する
4. docs/PRD.md に保存する
5. セルフレビューして「レビューして修正しますか？」と確認する
